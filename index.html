Over red waves of sand

Lo-fi adventure gaming in javascript

In all four directions my fortune it grows
 but over red waves of sand the wind never blows
 white sand at dusk turns orange before black
 over red waves of sand I will not come back

April 2013

This is a fighting/adventure game in a semi-ASCII style, written in javascript/jQuery. It's still in development but no commit should break basic functionality. If you run it, it'll work up to what I've written that day.

To run

Download the package and run index.html in your browser. Runs smoothly in Chrome, mostly smooth in Firefox and slow to sad in older versions of IE.

What works

Startup, movement, combat, opening doors, AI pathing/targeting, line-of-sight, spellcasting, win/loss messages. There's a lot to go.

What I'm working on

Better combat, multiple maps, by-player fog of war, more weapons/armor/items, custom customizing

Stuff I didn't write

jQuery (http://jquery.com/)
A* Pathing Algorithm in Javascript (https://github.com/bgrins/javascript-astar/)
Bresenham's Line Algorithm in Javascript (http://rosettacode.org/wiki/Bitmap/Bresenham's_line_algorithm#JavaScript)
